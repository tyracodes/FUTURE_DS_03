# FUTURE_DS_03
#  Marketing Funnel & Conversion Analysis Dashboard (task 3) 

This project is an interactive **Marketing Funnel & Conversion Analysis Dashboard** built using **Python and Streamlit**. It analyzes marketing campaign data to understand customer behavior, conversion performance, and funnel drop-offs.

---

##  Live Dashboard

 https://futureds03-tmi3kduzcx9rfir8ghhiza.streamlit.app/

---

##  Project Objective

The goal of this project is to analyze a marketing funnel and answer key business questions such as:

- Where do users drop off in the funnel?
- Which contact channels perform best?
- How does conversion vary across months?
- What factors influence customer conversion?

---

##  Key Features

- Interactive **Marketing Funnel Visualization**
- KPI Metrics:
  - Total Contacts
  - Total Customers
  - Conversion Rate
  - Average Call Duration
- Channel Performance Analysis
- Monthly Conversion Trends
- Campaign Effectiveness Analysis
- Customer Demographics (Age & Balance)
- Dynamic Filters (Month & Contact Type)

---

##  Key Insights

- Major drop-offs occur between early engagement and conversion stages.
- Certain contact channels (e.g., cellular/telephone) perform better than others.
- Conversion rates vary across different months, showing seasonal trends.
- Increasing campaign contact frequency does not always improve conversion.
- Customer age and balance influence likelihood of conversion.
